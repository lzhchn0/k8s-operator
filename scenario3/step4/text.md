# Build a framework of operator for redis-cluster


- Operator generated by kubebuilder

```
kubebuilder init   --plugins=go/v4   --repo blabla.com
```{{exec}}

```
kubebuilder create api \
  --kind Monitoring\
  --group rediscluster \
  --version v1 \
  --resource true \
  --controller true
```{{exec}}


```
make manifests
```{{exec}}



```
make install
```{{exec}}



```
make run
```{{exec}}
